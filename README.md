# Aprende Jugando con Papá 🚀

¡Bienvenido a **Aprende Jugando con Papá**! Este es un ecosistema interactivo de neuro-aprendizaje infinito diseñado especialmente para acompañar el crecimiento y aprendizaje de **Agustín**.

El proyecto está optimizado para funcionar directamente en cualquier navegador y dispositivo móvil como una **GitHub Page**.

## 🎮 Áreas de Aprendizaje

- 🔤 **Lectura**: Ruta progresiva que incluye:
  1. Vocales
  2. Sílabas Simples
  3. Sílabas de 3 letras
  4. Palabras Básicas
  5. Palabras Complejas
  6. Palabras Largas
- 🔢 **Matemáticas**: Práctica de antecesor y sucesor de números del 0 al 10.
- 🧮 **Conteo**: ¿Cuántos objetos hay en pantalla?
- 🧩 **Semántica**: Asociación interactiva de palabras con su respectivo emoji.
- 📐 **Geometría 2D y 3D**: Reconocimiento visual de formas geométricas básicas y figuras tridimensionales (cubo, pirámide, cilindro, esfera, etc.).

## ⚙️ Características Técnicas

- **Diseño Ultra-Moderno**: Interfaz con tema oscuro, gradientes dinámicos, fuentes elegantes (Nunito) y animaciones fluidas (micro-interacciones).
- **Algoritmo de Repetición Espaciada (Spaced Repetition)**: Las palabras que presentan mayor dificultad vuelven a aparecer más rápido para reforzar el aprendizaje dinámicamente.
- **Sistema de Logros**: Contador de racha actual (streaks) y total de sesiones guardadas localmente en el navegador (`localStorage`).
- **Interactividad Festiva**: Efecto de confeti y animaciones vibrantes al acertar las respuestas correctas.
- **Carga Instantánea**: Todo en un solo archivo HTML autónomo (`index.html`) sin dependencias externas pesadas.
